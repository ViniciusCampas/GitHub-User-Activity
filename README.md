ENG - GitNuh User Activity CLI

GitNuh User Activity CLI is a Python command-line application that allows users to track any GitHub user's activity. The program uses GitHub's public API to list the 
user's recent events, including Push, Issues, Watch, Pull Request Review, and Create Events.

The project is built with Python 3 and uses the requests library to make HTTP calls to the GitHub API. This makes the application lightweight, simple, and easy to run 
in any environment with Python installed.

The user only needs to enter the GitHub username when prompted. The program then retrieves the user's events and displays detailed information about each event type, 
including the affected repository, the performed action, and other relevant details depending on the event type.

Each event is displayed in a numbered list, providing a clear and organized view of the user's activity. If there are errors, such as a non-existent username or 
connection issues with the API, clear error messages are displayed.

This project is ideal for learning about API consumption, JSON data handling, loops, and command-line application development. It is suitable for beginners, students, 
or junior developers who want to explore GitHub data in a practical way.

PT-BR - GitNuh User Activity CLI

GitNuh User Activity CLI é uma aplicação de linha de comando desenvolvida em Python que permite acompanhar a atividade de qualquer usuário no GitHub. O programa acessa 
a API pública do GitHub para listar os eventos mais recentes do usuário, incluindo Push, Issues, Watch, Pull Request Review e Create Events.

O projeto foi desenvolvido utilizando Python 3 e a biblioteca requests para realizar chamadas HTTP à API do GitHub. Isso torna a aplicação simples, leve e fácil de 
executar em qualquer ambiente que tenha Python instalado.

O usuário precisa apenas digitar o nome de usuário do GitHub quando solicitado. O programa então coleta os eventos do usuário e exibe informações detalhadas sobre cada 
tipo de evento, incluindo o repositório afetado, a ação realizada e outros detalhes relevantes, dependendo do tipo de evento.

Cada evento é listado de forma numerada, permitindo uma visualização organizada da atividade do usuário. Em caso de erros, como usuário não encontrado ou problemas de 
conexão com a API, mensagens de erro claras são exibidas para o usuário.

Este projeto é ideal para aprendizado sobre consumo de APIs, manipulação de dados JSON, estruturas de repetição, e criação de aplicações de linha de comando. Pode ser 
usado por iniciantes, estudantes ou desenvolvedores em nível júnior que desejam explorar dados do GitHub de forma prática.
